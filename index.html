FAQ

How do I delete a pull request?

You can’t delete a PR. Close it on GitHub using “Close pull request,” then delete the branch. If it was merged, use the “Revert” button to undo.

What does a pull request do?

A pull request proposes changes and asks teammates to review before merging into the main branch. It shows the diff, lets others comment, and only merges when approved.
